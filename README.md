# cyber-center

Cyber Command Center is an interactive cybersecurity dashboard simulation that recreates a hacker-style control interface inside the browser.

The system features a terminal command interface, simulated network scanning, cyber attack visualization, system monitoring charts, database explorer, and a global cyber activity map. The interface is styled with a Matrix-inspired theme to provide an immersive cybersecurity environment.

This project is built using HTML, CSS, and JavaScript and is designed for educational purposes, UI experimentation, and cybersecurity-themed portfolio demonstrations.
⚠ Disclaimer:
This project is a simulation for educational and demonstration purposes only. It does not perform real hacking or network attacks.
